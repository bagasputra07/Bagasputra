1. Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Objek dibuat dari class yang berisi atribut dan method.

2. Contoh Class dan Object


class Car {

3. Kesimpulan

OOP membantu membuat program lebih terstruktur, mudah dikembangkan, dan lebih aman melalui konsep encapsulation.
