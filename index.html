1. PENGERTIAN OOP

Object Oriented Programming (OOP)...

OOP bertujuan untuk meningkatkan modularitas sistem...

Selain itu, OOP mendukung prinsip reusability...

Empat pilar utama yaitu Encapsulation, Abstraction, Inheritance, dan Polymorphism.

Paradigma ini menjadi fondasi utama dalam pengembangan aplikasi modern termasuk Flutter.

2. CLASS DAN OBJECT


class Car {
  String? model;
  String? color;

  void start() {
    print('Car started!');
  }

  void stop() {
    print('Car stopped!');
  }
}

Class Car memiliki atribut dan method yang menggambarkan objek mobil.

3. CONSTRUCTOR


class Car {
  String? model;
  String? color;

  Car(this.model, this.color);

  void start() {
    print('Car started!');
  }
}

Constructor memberikan nilai awal saat objek dibuat.

4. VALIDASI DATA


class Mahasiswa {
  String nama;
  String nim;
  double ipk;

  Mahasiswa(this.nama, this.nim, this.ipk) {
    if (ipk < 0.0 || ipk > 4.0) {
      throw ArgumentError('IPK harus 0.0 - 4.0');
    }
  }
}

Validasi memastikan IPK dalam rentang yang benar.

5. ENCAPSULATION


class Person {
  String _name;
  int age;

  Person(this._name, this.age);

  void setName(String name) {
    _name = name;
  }

  String getName() {
    return _name;
  }
}

Encapsulation membatasi akses langsung ke data menggunakan getter dan setter.

6. NAMED PARAMETER


class Produk {
  final String id;
  final String nama;
  final double harga;
  final int stok;
  final String kategori;

  Produk({
    required this.id,
    required this.nama,
    required this.harga,
    this.stok = 0,
    this.kategori = 'Umum',
  });
}

Named parameter membuat constructor lebih fleksibel dan aman.

7. KESIMPULAN

OOP membuat program lebih terstruktur, scalable, dan maintainable.
